# One-Click Invoice — Freelancer Invoice Generator

Small single-purpose web app to create invoices and export them as PDF. No backend required.

## Features
- Add invoice items (description, qty, rate)
- Automatic subtotal, tax and total calculation
- Export invoice as PDF (jsPDF)
- Simple, minimal UI — ready for GitHub Pages hosting

## How to run
1. Clone repository.
2. Open `index.html` in your browser.
3. Fill invoice fields, add items, click **Download PDF**.
