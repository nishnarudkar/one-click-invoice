# Product Overview

One-Click Invoice is a single-purpose web application for freelancers to generate professional PDF invoices quickly without requiring signup or a backend.

## Core Purpose
- Generate invoices with customizable line items (description, quantity, rate)
- Automatic calculation of subtotals, tax, and totals
- Export invoices as PDF files for client delivery

## Target Users
Freelancers and independent contractors who need a simple, no-frills invoicing solution without the overhead of full accounting software.

## Key Principles
- **Simplicity**: Single-page application with minimal UI
- **Privacy**: No data collection, no backend, no signup required
- **Portability**: Static files only, can be hosted anywhere (GitHub Pages, local file system)
- **Speed**: One-click PDF generation with pre-filled defaults
